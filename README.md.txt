# 🎓 Motivate AI Studio

**Motivate AI Studio** is a cloud-based app that helps coaching classes and educators create stunning, AI-generated motivational images and videos for students — every Monday, automatically!

## ✨ Features

- 📸 Generate motivational **images** and **videos** with professional templates
- 🎨 Select from multiple **color themes**, fonts, and backgrounds
- 🤖 Built-in **AI quote generator** (free and student-focused)
- 🖼️ Upload your **class logo**
- 📅 Automatic content generation every **Monday**
- 🧠 100+ included HD templates (both image and video)
- 🌐 100% cloud-based (no installation needed)
